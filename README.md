# Smart Pet Feeding System

该项目实现了一个智能宠物喂食系统，旨在解决主人不在家时宠物定时定量喂养的问题，并提升宠物饮食规律性与健康管理水平。

## 项目简介

系统以单片机为核心控制单元，通过时间调度、传感器输入和电机驱动，实现对宠物喂食和饮水的自动化管理。系统可预设多个喂食时段，自动启动步进电机完成投喂，并根据食物剩余情况进行补充控制。

## 核心功能

- 定时喂食：支持多个喂食时间点设置，自动执行投喂操作
- 食物监测：采用传感器实时检测剩余饲料量，避免投喂不足或浪费
- 饮水保障：集成水位传感器与水泵控制，确保饮水系统稳定可靠
- 远程监控：通过 WiFi 模块实现数据传输与手机端监控（如硬件端支持）
- 语音提示：提供语音反馈与状态提示，提高用户体验

## 技术要点

- 单片机控制：负责时钟管理、传感器采集、执行逻辑和设备驱动
- 步进电机驱动：精确控制投喂量和动作时序
- 传感器融合：压力/重量传感器、水位传感器等用于实时状态判断
- 系统逻辑：自动开关控制、定时调度、异常处理等关键模块
