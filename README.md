# Smart-Pet-Feeding-System
为了实现宠物自动喂食器，本设计旨在解决主人不在时宠物喂食的问题，还确保宠物饮食的规律性和健康。系统设计包括设定多个喂食时间点并发出喂食信号，以及根据食物剩余量自动控制喂食步进电机。此外，系统还集成水位传感器和水泵，保证宠物喂食时水源充足，所有数据可以通过WiFi模块传输到手机端进行实时监控和控制。核心技术工作包括单片机控制、自动开关和电机控制，同时集成了语音提示、时间设定、重量检测和水泵控制等关键系统。系统通过单片机持续监测设定的喂食时间点，并利用压力传感器实时检测食物剩余量，保证宠物在合适的时间和量进行喂食，实现了全自动化的宠物喂食管理。
